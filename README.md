<h1 align="center">HTML コーディング用<br />ボイラープレート</h1>

## Start 🚀

```bash
yarn start
```

Check http://localhost:3000/

## 技術

- EJS
- Gulp
- Sass
- TypeScript
- Webpack(js)
- Browser-sync
- Prettier
- Eslint
- [the-new-css-reset](https://github.com/elad2412/the-new-css-reset)
